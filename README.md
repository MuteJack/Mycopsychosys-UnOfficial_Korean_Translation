# Mycopsychosys-UnOfficial_Korean_Translation
마이코사이코시스(Mycopsychosys:Remastered) 게임의 비공식 한글 번역입니다.

## 적용 방법:
1. 게임의 /game/tl/russian 폴더에 .rpy파일들을 넣어주세요
2. 각 .rpy파일들을 열고, "translate korean" 부분을 모두 "translate russian"으로 일괄 변경합니다.
3. 게임을 켜고, russian을 선택해주세요.
4. russian외에도, 게임에서 지원하는 언어면 모두 적용 가능합니다.

본래 한국어 옵션을 추가하기 위해서는 실제 게임 내 원본 renpy스크립트를 수정해야합니다.
하지만 이를 위해서 수정된 스크립트를 함께 배포 시, 원본 스크립트 대다수가 포함(노출)되어 저작권 문제가 생길 수 있어 번역파일만을 첨부했습니다.
따라서, 버튼 ui 이미지 등은 덮어쓴 언어로 보이거나 게임 내 파일들을 Unpack 후 직접 수정해야하는 문제가 있습니다.

만약 이것이 싫으시거나, 간단한 if문 조차 모를 정도로 코딩에 대한 사전지식이 전혀 없으시다면
힌글번역, 추가 기능을 포함한 자동 패치 파일이 따로 있으니 아래 레포지터리를 참고해주세요.
